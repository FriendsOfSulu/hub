The Sulu Bulk Move Bundle provides a Symfony console command that moves all child pages from one navigation node to another, making it easy to restructure large parts of your page tree in a single operation. It is ideal for bulk reorganizations, migrations, or cleanups where many pages need to be relocated at once.
