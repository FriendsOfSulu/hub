A bundle for Sulu CMS that enhances the admin interface with reusable content types and powerful list transformers, making it easier to build rich editorial workflows. It combines and modernizes features from earlier Sulu bundles to provide a unified toolkit of field types, visual list renderers, and small admin UX improvements.
