The Sulu Snippet Tabs Bundle lets you define extra tabs for snippet forms via simple YAML configuration, so editors can manage additional structured data in a clean, tabbed UI. It stores the extra fields as extension content and integrates with the Sulu Snippet Manager Bundle, making it easy to extend snippets without custom PHP admin extensions.
